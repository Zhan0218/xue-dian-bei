# Route Map

## Dashboard
- `/dashboard` (DashboardPage) - 总览

## Users
- `/users` (UsersListPage) - 用户列表
- `/users/:id` (UserDetailPage) - 用户详情
- `/devices` (DevicesListPage) - 设备管理

## Policies & Plans
- `/policies` (PoliciesPage) - 拦截策略
- `/plan-templates` (PlanTemplatesPage) - 计划模板

## Support
- `/tickets` (TicketsListPage) - 工单列表
- `/tickets/:id` (TicketDetailPage) - 工单详情

## Analytics
- `/analytics/events` (EventsPage) - 事件列表

## Feature Flags
- `/feature-flags` (FeatureFlagsPage) - 功能开关

## System
- `/system/admin-users` (AdminUsersPage) - 管理员账号
- `/system/rbac` (RbacPage) - 角色与权限
- `/system/audit-logs` (AuditLogsPage) - 审计日志
- `/system/config` (ConfigCenterPage) - 配置中心
- `/system/ui` (UiSettingsPage) - UI 设置
